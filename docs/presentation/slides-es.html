Lo que Van a Dominar

  • TAOR — El ciclo de razonamiento que hace precisos a los agentes
  • LangChain — Cadenas, herramientas, memoria persistente
  • LangGraph — Grafos con estado y rutas condicionales
  • MCP Servers — Jira + Grafana + Git + DB al alcance del agente
  • RAG — El agente entiende TODO su codebase
  • CLAUDE.md — Sus convenciones, heredadas automaticamente

Aplicacion Real

  • Agente que revisa PRs — detecta bugs y sugiere fixes con codigo
  • Agente que genera documentacion — sincronizada con el codigo
  • Agente de respuesta a incidentes — logs, diagnostico, runbooks, post-mortem
  • Pipeline Jira-to-Code — del ticket al PR sin intervenir

Cada receta viene con codigo listo para produccion.
